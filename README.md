# Descripción del Proyecto

## Objetivo

Desarrollar una aplicación web utilizando el framework **OpenXava** que centralice la gestión de reparaciones, mantenimiento preventivo y control de inventario para órdenes de trabajo, permitiendo realizar un seguimiento en tiempo real del estado de los trabajos de mantenimiento y de la productividad de los mecánicos.

## Alcance

El proyecto contempla el desarrollo de un **prototipo web funcional** para la gestión de talleres mecánicos. El sistema integra los módulos necesarios para administrar clientes, vehículos, órdenes de trabajo, inventario de repuestos y la gestión financiera asociada a los servicios realizados.

## Módulos del Sistema

El sistema incluye los siguientes módulos:

- 👤 Gestión de Clientes
- 🚗 Gestión de Vehículos
- 📋 Gestión de Órdenes de Trabajo
- 📦 Gestión de Inventario
- 🔧 Asignación de Repuestos y Equipos
- 💰 Cálculo de Costos de las Órdenes de Trabajo
